Java Lab 03

Java Operators.

Typed notes for expressions and the main operator types used in Java programs.

Expression

An expression is a combination of variables, values, and operators that produces a result.

int a = b + c;
  • b + c is an expression.
  • + is an operator.
  • b and c are operands.

Types of Operators

No. Operator Type Operators Purpose
1 Arithmetic + - * / % Used for mathematical calculations.
2 Relational / Comparison > < >= <= == != Used to compare values and return a boolean result.
3 Logical && || ! Used to combine or reverse conditions.
4 Increment / Decrement ++ -- Used to increase or decrease a value by one.
5 Assignment = += -= *= /= %= Used to assign or update values.
6 Conditional / Ternary condition ? value1 : value2 Used to choose one value from two results.

Arithmetic Operators

Arithmetic operators are used for calculations.

int a = 50;
int b = 20;

System.out.println(a + b); // 70
System.out.println(a - b); // 30
System.out.println(a * b); // 1000
System.out.println(a / b); // 2
System.out.println(a % b); // 10
  • + adds values.
  • - subtracts values.
  • * multiplies values.
  • / gives quotient value.
  • % gives remainder value.

Relational / Comparison Operators

Relational operators compare two values and return a boolean result: true or false.

int a = 10;
int b = 20;

System.out.println(a > b);  // false
System.out.println(a < b);  // true
System.out.println(a == b); // false
System.out.println(a != b); // true
Operator Meaning
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to
== Equal to
!= Not equal to

Logical Operators

Logical operators work on boolean values and boolean expressions.

x y x && y x || y !x
true true true true false
true false false true false
false true false true true
false false false false true
  • && returns true only when both conditions are true.
  • || returns true when at least one condition is true.
  • ! reverses the boolean value.

Increment and Decrement Operators

Increment and decrement operators increase or decrease a value by one.

int a = 10;

System.out.println(a++); // 10
System.out.println(a);   // 11

System.out.println(++a); // 12

System.out.println(a--); // 12
System.out.println(a);   // 11

System.out.println(--a); // 10
  • a++ is post-increment: first prints the current value, then increases it.
  • ++a is pre-increment: first increases the value, then prints it.
  • a-- is post-decrement: first prints the current value, then decreases it.
  • --a is pre-decrement: first decreases the value, then prints it.

Assignment Operators

Assignment operators are used to assign or update values.

int a = 10;

a += 5; // same as a = a + 5
a -= 5; // same as a = a - 5
a *= 2; // same as a = a * 2
a /= 2; // same as a = a / 2
a %= 2; // same as a = a % 2

Conditional / Ternary Operator

The ternary operator is a short form of an if else statement.

variable = condition ? result1 : result2;
  • If the condition is true, result1 is given as output.
  • If the condition is false, result2 is given as output.
int a = 200;
int b = 100;

int x = (a > b) ? a : b;
System.out.println(x); // 200
