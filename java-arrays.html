Java Lab 06

Arrays.

Typed notes for storing multiple values, using indexes, reading values, and working with two-dimensional arrays.

What Is an Array?

An array is a collection of elements of the same data type.

  • An array can store multiple values inside a single variable.
  • All values in one array should be of the same data type.
  • Array index always starts from 0.

Types of Arrays

  1. Single-dimensional array: only rows.
  2. Multi-dimensional array: rows and columns.

Single-Dimensional Array

A single-dimensional array stores values in one row.

int a[] = {100, 200, 300, 400, 500};
Index Value
a[0] 100
a[1] 200
a[2] 300
a[3] 400
a[4] 500

Finding Length of an Array

length gives the total number of values in an array.

int a[] = {1, 2, 3, 4, 5};

System.out.println(a.length);
Output:
5

Read Specific Value from Array

To read one specific value, use the array name with its index.

int a[] = {10, 20, 30, 40, 50};

System.out.println(a[4]);
Output:
50

Read All Values Using Normal for Loop

A normal for loop uses index values to read each array value.

int a[] = {100, 200, 300, 400, 500};

for (int i = 0; i < a.length; i++) {
    System.out.println(a[i]);
}
  • i starts from 0 because array index starts from 0.
  • a.length gives the total number of values.
  • a[i] reads the current value from the array.

Read All Values Using Enhanced for Loop

An enhanced for loop reads each value directly without using index.

int a[] = {100, 200, 300, 400, 500};

for (int x : a) {
    System.out.println(x);
}

Two-Dimensional Array

A two-dimensional array stores data in rows and columns.

int a[][] = {
    {100, 200},
    {300, 400},
    {500, 600}
};
  • a[0][0] means row 0, column 0.
  • a[0].length gives column length.
  • a.length gives row length.
Column 0 Column 1
Row 0 100 200
Row 1 300 400
Row 2 500 600

Find Row and Column Length

int a[][] = {
    {100, 200},
    {300, 400},
    {500, 600}
};

System.out.println(a.length);    // row length
System.out.println(a[0].length); // column length

Read Specific Value from Two-Dimensional Array

int a[][] = {
    {100, 200},
    {300, 400},
    {500, 600}
};

System.out.println(a[2][1]);
Output:
600

Read All Values from Two-Dimensional Array

Use nested for loops to read rows and columns.

int a[][] = {
    {100, 200},
    {300, 400},
    {500, 600}
};

for (int r = 0; r < a.length; r++) {
    for (int c = 0; c < a[r].length; c++) {
        System.out.println(a[r][c]);
    }
}

Read All Values Using Enhanced for Loop

int a[][] = {
    {100, 200},
    {300, 400},
    {500, 600}
};

for (int row[] : a) {
    for (int value : row) {
        System.out.println(value);
    }
}
