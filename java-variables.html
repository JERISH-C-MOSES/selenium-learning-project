Variable and Data Type.

Typed notes for Java variables, constants, data types, and operators.

Variable

A variable is a named memory location used to store a value.

int a = 10;
  • int is the data type.
  • a is the variable name.
  • 10 is the value stored in the variable.

Variable Naming Rules

  1. Variable names can contain letters, digits, underscores, and dollar signs.
  2. A variable name should usually begin with a letter.
  3. A variable name must not contain spaces.
  4. A variable name can also begin with $ or _.
  5. Variable names are case-sensitive, so myVar and myvar are different.
  6. Reserved Java keywords like int, class, and boolean cannot be used as variable names.

Correct and Wrong Variable Names

Example Status Reason
int ab2; Correct Digits are allowed after letters.
int a_b; Correct Underscore can be used instead of space.
int $value; Correct Dollar sign is allowed.
int 2ab; Wrong Variable name cannot start with a digit.
int a b; Wrong Spaces are not allowed in variable names.
int a-b; Wrong Hyphen is treated like a minus operator.
int class; Wrong class is a Java keyword.

Constant

A variable declared with final becomes a constant, which means the value cannot be changed.

final int a = 15;
  • final makes the value fixed.
  • After assigning 15, the value of a cannot be changed.
