Java Lab 04

Conditional Statement.

Decision-making statements used to execute code only when a condition matches.

What Is a Conditional Statement?

A conditional statement checks a condition and runs a block of code based on the result.

  • If the condition is true, the statement inside that block will execute.
  • If the condition is false, Java will skip that block or move to another condition.
  • The condition usually returns a boolean value: true or false.

Types of Conditional Statements

  1. if
  2. if else
  3. nested if else
  4. else if ladder
  5. switch case

1. if Statement

The if statement executes only when the condition is true.

if (condition) {
    statement;
}
int age = 25;

if (age >= 18) {
    System.out.println("Eligible for vote");
}
  • age >= 18 is the condition.
  • If the condition is true, Eligible for vote will be printed.
  • If the condition is false, nothing will be printed because there is no else block.

2. if else Statement

The if else statement gives two paths: one for true and one for false.

if (condition) {
    statement;
} else {
    statement;
}
int age = 15;

if (age >= 18) {
    System.out.println("Eligible for vote");
} else {
    System.out.println("Not eligible for vote");
}
  • If age >= 18 is true, the first block executes.
  • If age >= 18 is false, the else block executes.

3. Nested if else

Nested if else means writing one if else statement inside another if else statement.

if (condition1) {
    statement;
} else if (condition2) {
    statement;
} else if (condition3) {
    statement;
} else {
    statement;
}
int number = 0;

if (number > 0) {
    System.out.println("Positive number");
} else if (number < 0) {
    System.out.println("Negative number");
} else {
    System.out.println("Zero");
}
  • First Java checks whether the number is positive.
  • If that is false, Java checks whether the number is negative.
  • If both conditions are false, the else block prints Zero.

4. Else-if Ladder

An else-if ladder checks multiple conditions one by one from top to bottom.

int mark = 72;

if (mark >= 90) {
    System.out.println("Grade A");
} else if (mark >= 75) {
    System.out.println("Grade B");
} else if (mark >= 50) {
    System.out.println("Grade C");
} else {
    System.out.println("Fail");
}
  • Java checks the first condition.
  • If it is false, Java moves to the next condition.
  • When one condition becomes true, that block executes and the remaining conditions are skipped.

5. Switch Case

switch is used when one variable can match different fixed values.

switch (variable) {
    case value1:
        statement;
        break;
    case value2:
        statement;
        break;
    case value3:
        statement;
        break;
    default:
        statement;
}
int week = 1;

switch (week) {
    case 1:
        System.out.println("Sunday");
        break;
    case 2:
        System.out.println("Monday");
        break;
    case 3:
        System.out.println("Tuesday");
        break;
    case 4:
        System.out.println("Wednesday");
        break;
    case 5:
        System.out.println("Thursday");
        break;
    case 6:
        System.out.println("Friday");
        break;
    case 7:
        System.out.println("Saturday");
        break;
    default:
        System.out.println("Invalid week number");
}
  • switch checks the value of one variable.
  • case is used for each possible value.
  • break stops the switch after the matching case runs.
  • default runs when no case value matches.
