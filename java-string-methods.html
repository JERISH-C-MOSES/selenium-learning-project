Java Lab 07

String Methods.

Typed notes for commonly used Java string methods with syntax, examples, and important points.

What Are String Methods?

String methods are built-in methods used to perform operations on string values.

  • A string can store words, names, sentences, numbers written inside double quotes, and spaces.
  • Most string methods return a new value. The original string value is not changed.
  • String indexes start from 0.

1. Find the Length of a String

length() is used to find the total number of characters in a string.

stringname.length();

Note: It will consider spaces also.

String s = "Jerish C Moses";
int l = s.length();

System.out.println(l);
Output:
15

2. Join Two Strings

concat() joins two strings together.

string1.concat(string2);
String s1 = "Jerish C";
String s2 = "Moses";

String s = s1.concat(s2);
System.out.println(s);
Output:
Jerish CMoses

To keep a space between the two strings, add a space manually.

String s = s1.concat(" ").concat(s2);
System.out.println(s);
Output:
Jerish C Moses

The plus operator can also join strings.

String s3 = "Ganesh";

System.out.println(s1 + s2 + s3);
System.out.println(s1.concat(s2).concat(s3));

3. Remove Spaces Before and After Words

trim() removes extra spaces from the beginning and ending of a string.

stringname.trim();

Note: Spaces inside the sentence will not be removed.

String s1 = "   Jerish C Moses   ";

String s = s1.trim();
System.out.println(s);
Output:
Jerish C Moses

4. Get a Specific Character Using Index

charAt() returns the character present at a specific index.

stringname.charAt(index);

Note: Index starts with 0.

String s = "Welcome";

char c = s.charAt(0);
System.out.println(c);
Output:
W

5. Check Whether One String Contains Another String

contains() checks whether the main string contains the given text.

stringname.contains("text");

Note: It returns a Boolean value and is case sensitive.

String s1 = "Welcome to Earth";

boolean result = s1.contains("Earth");
System.out.println(result);
Output:
true
boolean result = s1.contains("earth");
System.out.println(result);
Output:
false
String s1 = "Welcome";
String s2 = "com";

if (s1.contains(s2)) {
    System.out.println("Yes");
} else {
    System.out.println("No");
}
Output:
Yes

6. Check Whether Two Strings Are Equal

equals() compares two strings and checks whether both are exactly equal.

string1.equals(string2);

equals() is case sensitive.

Note: When you use == with strings in Java, it does not compare the content of the string. It compares memory location references.

String s1 = "Welcome";
String s2 = "welcome";

System.out.println(s1.equals(s2));
Output:
false

equalsIgnoreCase() compares strings without checking uppercase and lowercase differences.

String s1 = "Welcome";
String s2 = "welcome";

System.out.println(s1.equalsIgnoreCase(s2));
Output:
true

7. Replace Method

replace() replaces an old value with a new value.

stringname.replace(oldValue, newValue);
String s = "Welcome to Java";

String updated = s.replace("Java", "Selenium");
System.out.println(updated);
Output:
Welcome to Selenium
