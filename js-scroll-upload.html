Help Center

Scrolling Pages

Important Notes

Note: The scroll bar in the browser is not a web element. It is part of the browser.

Scrolling can be done using JavascriptExecutor.

Syntax

Scroll Down by Pixel

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("window.scrollBy(0,500)");
Explanation
  • JavascriptExecutor js = (JavascriptExecutor) driver; converts driver into JavascriptExecutor.
  • window represents browser window.
  • scrollBy() scrolls from current position.
  • 0 means no horizontal movement.
  • 500 scrolls down by 500 pixels.

Scroll Up by Pixel

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("window.scrollBy(0,-500)");
Explanation
  • scrollBy() scrolls from current position.
  • 0 means no horizontal movement.
  • -500 scrolls up by 500 pixels.

Scroll Element Into View

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("arguments[0].scrollIntoView()", webelement);
Explanation
  • arguments[0] represents the web element.
  • scrollIntoView() scrolls until element becomes visible.
  • webelement passed after comma becomes arguments[0].

Scroll to Bottom of Page

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("window.scrollTo(0,document.body.scrollHeight)");
Explanation
  • scrollTo() moves to exact position.
  • 0 is horizontal position.
  • document.body.scrollHeight gives total webpage height.
  • Page scrolls directly to bottom.

Scroll to Top of Page

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("window.scrollTo(0,0)");
Explanation
  • scrollTo() moves to exact position.
  • First 0 is horizontal position.
  • Second 0 is top of webpage.

Horizontal Scroll

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("window.scrollBy(500,0)");
Explanation
  • 500 moves horizontally to right.
  • 0 means no vertical scrolling.
  • scrollBy() scrolls relative to current position.

Print How Many Pixels the Page Has Moved

JavascriptExecutor js = (JavascriptExecutor) driver;

System.out.println(js.executeScript("return window.pageYOffset;"));
Explanation
  • JavascriptExecutor js = (JavascriptExecutor) driver; converts driver into JavascriptExecutor.
  • executeScript() executes JavaScript code inside browser.
  • return sends value back to Java.
  • window.pageYOffset returns how many pixels the page has moved vertically.
  • System.out.println() prints the pixel value in console.

Full Scrolling Example

// Scroll Down by Pixel
js.executeScript("window.scrollBy(0,2200)");
Thread.sleep(2000);
System.out.println(js.executeScript("return window.pageYOffset;"));

Help Center

JavascriptExecutor

Where JavascriptExecutor Fits

JavascriptExecutor is connected with browser driver classes through RemoteWebDriver. WebDriver is the main browser automation contract, and driver classes such as ChromeDriver, EdgeDriver, FirefoxDriver, SafariDriver, OperaDriver, and InternetExplorerDriver work through that driver hierarchy.

Why We Use It

  1. This method is useful when Selenium's normal actions fail, especially in cases like ElementClickInterceptedException.
  2. When Selenium executes normal methods like click(), internally the browser uses JavaScript to communicate and interact with the web element.
  3. Selenium methods such as click() internally communicate with the browser to perform user-like actions.
  4. Instead of depending only on Selenium's normal click method, we can directly execute JavaScript code using JavascriptExecutor to interact with the element.

To call JavascriptExecutor, we use executeScript().

Note 1

When using WebDriver driver = new ChromeDriver();, use upcasting: 

JavascriptExecutor js = (JavascriptExecutor) driver;

This is needed because WebDriver and JavascriptExecutor are separate interfaces.

Note 2

When using ChromeDriver driver = new ChromeDriver();, there is no need for upcasting: 

JavascriptExecutor js = driver;

ChromeDriver driver = new ChromeDriver(); does not need upcasting because the reference variable type (ChromeDriver) and the object type (ChromeDriver) are the same, whereas upcasting happens only when a child class object is stored in a parent class reference like WebDriver driver = new ChromeDriver();.

Syntax

Click Using JavaScript

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("arguments[0].click();", webelement);

SendKeys Using JavaScript

JavascriptExecutor js = (JavascriptExecutor) driver;

js.executeScript("arguments[0].setAttribute('value','Jerish')", webelement);

Syntax Breakdown

Click Using JavaScript

  1. JavascriptExecutor js = (JavascriptExecutor) driver;

    Converts the driver into JavascriptExecutor.

  2. executeScript() is used to execute JavaScript code inside the browser.

  3. arguments[0]

    Represents the web element passed after the comma.

  4. arguments[0].click();

    Performs click operation on the element using JavaScript.

  5. This method is useful when normal Selenium click() does not work properly, especially during ElementClickInterceptedException.

  6. The webelement written after the comma is passed as an argument to the JavaScript code, so arguments[0] refers to that web element.

SendKeys Using JavaScript

  1. JavascriptExecutor js = (JavascriptExecutor) driver;

    Converts the driver into JavascriptExecutor.

  2. executeScript() is used to execute JavaScript code inside the browser.

  3. arguments[0]

    Represents the web element passed after the comma.

  4. arguments[0].setAttribute('value','Jerish')

    Sets the value attribute of the textbox to Jerish.

  5. setAttribute() is a JavaScript method used to change an attribute value of an element.

  6. This method is useful when normal Selenium sendKeys() does not work properly or when we want to set a value directly.

  7. The webelement written after the comma is passed as an argument to the JavaScript code, so arguments[0] refers to that web element.

Full Login Example Using JavascriptExecutor

public class JavascriptExecutorTest {

    public static void main(String[] args) {

        WebDriver driver = new ChromeDriver();

        driver.get("file:///C:/Users/hello/OneDrive/Documents/New%20project/selenium-practice-website/index.html");
        driver.manage().window().maximize();

        WebElement input = driver.findElement(By.xpath("//*[@id=\"username\"]"));

        JavascriptExecutor js = (JavascriptExecutor) driver;

        js.executeScript("arguments[0].setAttribute('value','JERISH')", input);

        WebElement pass = driver.findElement(By.xpath("//*[@id=\"password\"]"));

        js.executeScript("arguments[0].setAttribute('value','Jerish@123')", pass);

        WebElement submit = driver.findElement(By.xpath("//*[@id=\"login-button\"]"));

        js.executeScript("arguments[0].click()", submit);
    }
}
