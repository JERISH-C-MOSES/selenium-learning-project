Mouse Actions Notes

Mouse Actions

Mouse actions can be done using a predefined class called Actions in Selenium.

Actions - predefined class provided in Selenium

Mouse Hover Action

In Selenium, a mouse hover action means moving the mouse pointer over a web element without clicking it.

Syntax

Actions act = new Actions(driver);

act.moveToElement(element).build().perform();

Difference Between build() and perform()

.build()
  • Builds the action.
  • Creates a complete action sequence.
.perform()
  • Executes the action.

Mouse Hover Example

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.interactions.Actions;

public class HoverAndClickExample {

    public static void main(String[] args) {

        WebDriver driver = new ChromeDriver();

        driver.get("https://www.amazon.in");
        driver.manage().window().maximize();

        // Locate main menu
        WebElement accountMenu =
                driver.findElement(By.id("nav-link-accountList"));

        // Create Actions object
        Actions act = new Actions(driver);

        // Hover on Accounts and Lists
        act.moveToElement(accountMenu)
                .build()
                .perform();

        // Locate submenu after hover
        WebElement orders =
                driver.findElement(By.linkText("Your Orders"));

        // Click submenu
        act.click(orders)
                .build()
                .perform();
    }
}

Right Click in Selenium Java

Right click is also called:

Context Click

It is used to open the browser context menu or options on an element.

Example

  • Right click on a button.
  • Right click on a link.
  • Right click on a file or folder option.

Definition

Right click action is performed using:

contextClick()

It is a method from the Selenium Actions class.

Syntax

Right click on specific element:

Actions act = new Actions(driver);

act.contextClick(element)
        .build()
        .perform();

Right Click Example

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.interactions.Actions;

public class RightClickExample {

    public static void main(String[] args) {

        WebDriver driver = new ChromeDriver();

        driver.get("https://demo.guru99.com/test/simple_context_menu.html");
        driver.manage().window().maximize();

        // Locate button
        WebElement button =
                driver.findElement(By.xpath("//span[text()='right click me']"));

        // Create Actions object
        Actions act = new Actions(driver);

        // Perform right click
        act.contextClick(button)
                .build()
                .perform();
    }
}

Drag and Drop in Selenium Java

Definition

Drag and drop is performed using:

dragAndDrop()

It is a method from the Selenium Actions class.

Syntax

Actions act = new Actions(driver);

act.dragAndDrop(sourceElement, targetElement)
        .build()
        .perform();

Example

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.interactions.Actions;

public class DragAndDropExample {

    public static void main(String[] args) {

        WebDriver driver = new ChromeDriver();

        driver.get("https://demoqa.com/droppable");
        driver.manage().window().maximize();

        // Source element
        WebElement source =
                driver.findElement(By.id("draggable"));

        // Target element
        WebElement target =
                driver.findElement(By.id("droppable"));

        // Create Actions object
        Actions act = new Actions(driver);

        // Perform drag and drop
        act.dragAndDrop(source, target)
                .build()
                .perform();
    }
}

Actions Vs Action

Actions -- class, used to perform mouse actions.
Action  -- interface, used to store created actions.
