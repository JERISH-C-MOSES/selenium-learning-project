Loop and Jumping Statement.

Typed notes for repeating code and controlling loop flow using break and continue.

Types of Loops

  1. while loop
  2. do while loop
  3. for loop
  4. enhanced for loop

A loop is used when we want to execute the same statement again and again.

1. while Loop

A while loop checks the condition first. If the condition is true, then the statements are executed.

initialization;

while (condition) {
    statement;
    increment/decrement;
}
int i = 1;

while (i <= 10) {
    System.out.println(i);
    i++;
}
  • This prints numbers from 1 to 10.
  • If the condition is false at the beginning, the loop will not execute even once.

2. Printing Odd and Even Numbers

We can combine while with if else to check odd and even numbers.

int i = 1;

while (i <= 10) {
    if (i % 2 == 0) {
        System.out.println(i + " Even");
    } else {
        System.out.println(i + " Odd");
    }
    i++;
}

3. do while Loop

A do while loop executes the statement first, then checks the condition.

do {
    statement;
    increment/decrement;
} while (condition);
int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 10);
  • This prints numbers from 1 to 10.
  • In a do while loop, the statement executes at least once.

Difference Between while and do while

while do while
Checks condition before executing the statement. Executes the statement first, then checks condition.
If condition is false, output may be blank. If condition is false, statement still runs one time. 
int i = 10;

while (i <= 5) {
    System.out.println(i);
    i++;
}

// Output: blank
int i = 10;

do {
    System.out.println(i);
    i++;
} while (i <= 5);

// Output: 10

4. for Loop

A for loop keeps initialization, condition, and increment/decrement in one line.

for (initialization; condition; increment/decrement) {
    statement;
}
for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}
  • This prints numbers from 1 to 10.
  • The loop checks the condition before every iteration.

5. Enhanced for Loop

An enhanced for loop is mainly used to read values from arrays or collections.

for (datatype variable : collection) {
    statement;
}
int[] numbers = {1, 2, 3, 4};

for (int number : numbers) {
    System.out.println(number);
}

Jumping Statements

  1. break
  2. continue

break Statement

break is used to stop the loop and come out of it.

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
}
Output:
1
2
3
4

When i == 5, the condition becomes true and the loop stops directly.

continue Statement

continue skips the current iteration and moves directly to the next iteration.

for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        continue;
    }
    System.out.println(i);
}
Output:
1
2
3
4
6
7
8
9
10

When i == 5, only that iteration is skipped. The loop continues from the next value.
